# Project: Camelot Nuxt Layer

## Overview
A Nuxt layer providing the CamelotUI component library.

## Core Technologies
- Nuxt 4
- Vue 3 (Composition API)
- Tailwind CSS v4
- Pinia
- TypeScript
- date-fns

## Current Modules
- `app/components/Camelot/`: Core UI components with native Material, Cupertino, Sci-Fi HUD, and Cyberpunk styles.
- `app/components/Camelot/Internal/`: Internal sub-components (e.g., Calendar).
- `app/composables/useCamelotTheme`: State manager for global style selection.
