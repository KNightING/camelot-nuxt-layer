# Project: Camelot Nuxt Layer

## Overview
A Nuxt layer providing the CamelotUI component library.

## Core Technologies
- Nuxt 4
- Vue 3 (Composition API)
- Tailwind CSS v4
- Pinia
- TypeScript
- date-fns

## Current Modules
- `app/components/Camelot/`: Core UI components.
- `app/components/Camelot/Internal/`: Internal sub-components (e.g., Calendar).
