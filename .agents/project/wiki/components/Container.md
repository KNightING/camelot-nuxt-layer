# Container

> 帶有 Camelot 自訂捲軸的容器，將內容包裹於捲動區域中。

**匯入名稱**：`CamelotContainer`（Nuxt 自動匯入）

## Props
| Prop | 型別 | 預設 | 說明 |
| :--- | :--- | :---: | :--- |
| `horizontal` | `boolean` | — | 是否開啟水平滾動模式 |

## Slots
| Slot | 作用域參數 | 說明 |
| :--- | :--- | :--- |
| `default` | — | 容器內容 |

## 備註
- 內部使用 `CamelotScrollbar`，並將 `horizontal` 傳遞給它。
