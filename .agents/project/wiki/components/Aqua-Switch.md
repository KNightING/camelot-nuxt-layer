# Aqua Switch

> Aqua（毛玻璃）風格開關切換元件。內部/主題實作。

**匯入名稱**：`CamelotAquaSwitch`

## Props
| Prop | 型別 | 預設 | 說明 |
| :--- | :--- | :---: | :--- |
| `disabled` | `boolean` | `false` | 停用開關。 |

## Emits
| 事件 | 參數 | 說明 |
| :--- | :--- | :--- |
| `change` | `(checked: boolean)` | 切換狀態改變時觸發。 |

## v-model
| Model | 型別 | 說明 |
| :--- | :--- | :--- |
| `v-model` | `boolean` | 開關狀態。預設 `false`。 |

## 備註
- 停用時點擊不會切換。
